How This Works

The Proxy

PearProxy works by proxying all of the HTTP requests from your computer through the server. The server basically acts as a drop-in replacement for the website you are trying to get to, and tries it's best to seem exactly like the original site. It is 100% free and works without configuring proxy settings, because you're techincally only visiting the PearProxy website, and not the actual destination.

The DataURL

DataURLS are special website adresses that render whatever code is in them as the webpage. You don't need to connect to any server to load a dataurl (but the styles from TailwindCSS may be broken.)
